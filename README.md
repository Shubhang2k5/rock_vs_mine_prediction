# Rock vs Mine Prediction

A Machine Learning project that classifies SONAR signals as **Rock** or **Mine** using supervised learning.

## Dataset
- SONAR dataset
- 60 numerical features per sample
- Binary classification (Rock / Mine)

## Tech Stack
- Python
- NumPy
- Pandas
- Scikit-learn
- Jupyter Notebook

## Model
- Logistic Regression (or mention actual model)
- Train/Test Split
- Accuracy: ~76%
